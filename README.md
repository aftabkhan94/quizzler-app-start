This is a quizzler app. 

You can create a quick quiz on this app where it pulls data from this website: https://opentdb.com/
using API keys

if you wish to customize the category or number of questions in the quiz then you can edit the parameters dictionary in the data.py file.

Note: This quiz app is customized to ask only True or False question. Therefore, please do not change the boolean value from the parameters dictionary.

Happy Quizzing!
